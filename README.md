# Persistent Homology on Directed Graphs

This repository contains code for Dowker sink filtration (Chowdhury, Mémoli, 2018) and Walk-Length filtration (Muñoz, Munch, Khasawneh, 2025) for weighted directed graphs, as well as an application on simulated hippocampal networks.

For more details, go to _The Walk-Length Filtration for Persistent Homology on Weighted Directed Graphs - Muñoz, Munch, Khasawneh (2025)_ [here](https://arxiv.org/abs/2506.22263).
